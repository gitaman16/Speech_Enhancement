This folder stores model weight files.

Files expected here:
  unet_final_continued.pt    ← Original PyTorch checkpoint (copy from your original project)
  unet_tf_weights.h5         ← Generated by:  python weights/convert_weights.py
  checkpoint_epoch_*.h5      ← Generated by:  python train.py  (every 5 epochs)

IMPORTANT:
  Never modify or delete unet_final_continued.pt.
  It is the original trained checkpoint and cannot be recovered without retraining.
